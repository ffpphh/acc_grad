# acc_grad
Code for generating the numerical results (including experiment and plotting scripts) for the paper **"Understanding Accelerated Gradient Methods: Lyapunov Analyses and Hamiltonian-assisted Interpretations** by Penghui Fu and Zhiqiang Tan

# Files
## main.R
Contain functions of first-order optimization algorithms (including gradient descent, Nesterov's accelerated gradient method, and the algorithms proposed in this paper) as well as objective functions.
## simu.Rmd
R Markdown file to conduct numerical experiments.
